## Getting Started

Please look at the project documentation that has details and screenshots of the web application- [[PDF](https://github.com/Akshayalnyn/Fair-Compute/blob/main/Faircompute%20submission.pdf)]([https://faircompute.com/](https://github.com/Akshayalnyn/Fair-Compute/blob/main/Faircompute%20submission.pdf))

### Prerequisites

Install all the required packages from requirement.txt file.

### Installation

1. Get a free API Key at [https://faircompute.com/](https://faircompute.com/) for the LLava endpoint (llava:7b-v1.6-mistral-q5_K_M) and at [https://huggingface.co/](https://huggingface.co/) for Adrenex/fastgen endpoint.
2. Clone the repo

   ```sh
   git clone https://github.com/Akshayalnyn/Fair-Compute
   ```

3. Enter your API and credentials
   ```js
   API_KEY = "ENTER YOUR API";
   ```
